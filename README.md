# 🔐 AI SOC Threat Hunting Platform

## 🚀 Overview
This project is an advanced AI-powered SOC (Security Operations Center) platform designed to simulate real-world cybersecurity operations.

It combines phishing detection, threat intelligence, user authentication, and interactive dashboards into a single platform.

---

## 🔥 Features

### 🔐 Authentication
- Secure login/signup with password hashing
- Session timeout protection
- Forgot password system

### 🧠 AI Threat Detection
- Phishing detection using Machine Learning
- Risk scoring system

### 🌐 Threat Intelligence
- URL extraction and analysis
- VirusTotal API integration (optional)
- IOC-style detection

### 📊 Dashboard
- Plotly-based interactive charts
- Attack trends visualization
- Alert monitoring system

### 👑 Admin Panel
- View all users
- Monitor alerts
- System control

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- SQLite
- Plotly
- Requests (API)
